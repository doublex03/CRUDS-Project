## A lightweight web application built to manage product data using the core CRUDS principles (Create, Read, Update, Delete, Search).

This project demonstrates essential front-end data manipulation techniques and local storage implementation.

##  Features

* *Create:* Seamlessly add new product records.
* *Read:* Display and retrieve all stored product data in a dynamic table.
* *Update:* Modify existing product details within the table view.
* *Delete:* Remove single product records or wipe all data with a mass delete option.
* *Search:* Instant filtering of products by *Name* or *Category*.
* *Local Persistence:* Data is stored using the browser's *Local Storage*, ensuring persistence across sessions.
